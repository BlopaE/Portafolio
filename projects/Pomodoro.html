Reloj Pomodoro

con JAVA

Una de las técnicas de estudio más usadas y recomendadas, es el reloj pomodoro, el cual consiste en un temporizador de 25 minutos de trabajo sin concentraciones, seguido de 5 minutos de descanso, a estos 30 minutos se les llama pomodoro, y al cumplir 4 de estos, se podrá descansar 25 minutos. Este reloj implementa un reloj con dicha funcionalidad, haciendo uso del lenguaje JAVA

Ver Proyecto

Para la creación de este reloj, se utilizó la tecnologia JAVA Swing, la cual permite utilizar ventanas y componentes para una interfaz de usuario amigable, también se hizo uso de Threads, para tener multiples procesos como el reloj en funcionamiento. El programa implementa una ventana de registros, donde se pueden consultar las sesiones realizadas, las cuales persisten en un archivo de manera local, al igual que las configuraciones como: El tiempo de cada seccion y el tema. Dicho proyecto me ha sido de gran ayuda a mi y a mis compañeros que me han pedido que lo distribuya, lo considero uno de mis mejores proyectos JAVA
